# Library Management System (In development phase)

A desktop-based Library Management System built with Java, focusing on efficient data structures, custom algorithms, and a clean separation of concerns using Layered Architecture.

## Features

* **Book Registration & Management:** it's not a full CRUD operations but you can Create, Read and Modify books.
* **Graphical User Interface (GUI):** intuitive desktop interface built with Java Swing (development phase)
* **Local Persistence (while app running):** For this project i used the data structures for storage the info, in the future can migrate to DB 
* **Custom Algorithms:** Implementation of core searching and sorting algorithms for handling library records efficiently (development phase).

---

## Architecture & Design

The project follows a **Layered Architecture** pattern to ensure high maintainability, low coupling, and clear separation of concerns (without data layer):

1. **Presentation Layer (GUI):** Handles user interactions and input validation using Java Swing components.
2. **Business Logic Layer (BLL / Service):** Contains the core application rules, algorithms, and data processing logic.

### Data Structures Utilized
* Optimized collection types chosen specifically to manage memory references and entities effectively within the application lifecycle.
